# 網路封包系統

> 所屬：[L1-fastdesign](../index.md)

## 概述

Fast Design 擁有獨立的 Forge 網路頻道（`FdNetwork`），與 API 層的 `BRNetwork` 完全分離。封包涵蓋選取區域同步、全息投影控制、CAD 畫面開啟、GUI 操作代理、貼上預覽與放置等功能。封包方向包含 Server-to-Client（S→C）與 Client-to-Server（C→S）兩類。
