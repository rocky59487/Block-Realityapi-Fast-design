# 📄 節點畫布 UI

## 📖 內容
NodeCanvasScreen 是節點編輯器主畫面，包含 NodeWidgetRenderer、WireRenderer、NodeSearchPanel、NodeTooltipRenderer、BoxSelectionHandler、NodeCanvasUndoManager。各種 Inline 控制項（InlineSlider、InlineDropdown、InlineColorPicker 等）提供節點內嵌參數編輯。
