# 📄 工具節點（Tool Nodes）

## 📖 內容
工具節點分為：placement（放置如 BlueprintPlaceNode、GhostBlockNode、BuildModeNode、QuickPlacerNode）、selection（選擇如 SelectionFilterNode、BrushConfigNode、ToolMaskNode、SelectionVizNode）、input（輸入如 KeyBindingsNode、MouseConfigNode、GamepadConfigNode）、ui（介面如 HUDLayoutNode、CrosshairNode、RadialMenuNode、ThemeColorNode）。
