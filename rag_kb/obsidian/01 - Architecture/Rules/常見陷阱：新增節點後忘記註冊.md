# 📄 常見陷阱：新增節點後忘記註冊

## 📖 內容
新增 Node 子類別後，必須在 NodeRegistry 中註冊，否則儲存/載入節點圖時會找不到對應類別。若新增大量節點後忘記註冊，可執行 python fix_registry.py 自動掃描並補上。
