# 🧩 com.blockreality.api.client.render.ui.BRBlueprintPreview

> [!info] 摘要
> 客戶端藍圖預覽系統 — Axiom 風格即時預覽 + SimpleBuilding 風格快速操作。  功能： - 載入藍圖為半透明「幽靈方塊」預覽 - 即時旋轉 (Y 軸 90° 步進) / 鏡像 (X/Y/Z) - 預覽隨玩家準星移動（吸附到方塊網格） - 碰撞檢測（可選：顯示重疊方塊為紅色） - 快速放置確認（Enter / 右鍵） - 多藍圖快速切換（滾輪或快捷鍵）  渲染部分僅提供資料，實際 GL 繪製由 BREffectRenderer / pipeline 負責。  @author Block Reality Team @version 1.0
