# 🧩 com.blockreality.api.client.render.animation.Bone

> [!info] 摘要
> 骨骼層級系統 - 用於 Minecraft Forge 1.20.1 的角色和方塊實體骨骼動畫。 靈感源自 GeckoLib 的 GeoBone 層級骨骼系統，支持骨骼矩陣計算和蒙皮動畫。  該系統允許： 1. 構建骨骼層級結構（父子關係） 2. 計算世界空間變換矩陣 3. 應用動畫夾片到骨骼 4. 混合多個動畫夾片 5. 生成用於著色器上傳的蒙皮矩陣  @author Block Reality Animation System
