# 🧩 com.blockreality.api.spi.BatchBlockPlacer

> [!info] 摘要
> 高效批次方塊放置工具。  將方塊放置加入佇列而不觸發個別物理事件， 然後一次性刷新全部。抑制個別放置事件， 並在結束時發送單一批次完成事件。  這是 Fast Design 模組的核心工具， 用於實現高效的結構設計操作。
