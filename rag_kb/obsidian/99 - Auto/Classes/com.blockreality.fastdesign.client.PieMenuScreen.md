# 🧩 com.blockreality.fastdesign.client.PieMenuScreen

> [!info] 摘要
> 輻射狀快捷輪盤 — Fast Design v2.0 Feature 3  按住 Alt 鍵呼出環狀 GUI，利用滑鼠甩動方向快速選擇操作。 取代全螢幕的 ControlPanelScreen，大幅提升盲操效率。  設計參考：DOOM Eternal 武器輪盤 + 原神元素切換輪。  操作方式： - 按住 Alt → 彈出輪盤 - 移動滑鼠到扇區 → 高亮 - 放開 Alt → 執行選中操作
