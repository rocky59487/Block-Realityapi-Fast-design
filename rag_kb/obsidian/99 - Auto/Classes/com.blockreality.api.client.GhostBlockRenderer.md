# 🧩 com.blockreality.api.client.GhostBlockRenderer

> [!info] 摘要
> Ghost Block Renderer — draws semi-transparent preview cubes at proposed placement positions.  Usage: 1. Call {@link #setPreview(List)} from client tick to update the preview list 2. Call {@link #onRenderLevel(RenderLevelStageEvent)} from RenderLevelStageEvent 3. Call {@link #clearPreview()} when the player cancels or confirms placement  Rendering uses POSITION_COLOR with alpha blending, no texture
