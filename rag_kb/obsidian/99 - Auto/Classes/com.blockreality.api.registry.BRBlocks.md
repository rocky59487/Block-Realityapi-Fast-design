# 🧩 com.blockreality.api.registry.BRBlocks

> [!info] 摘要
> Block Reality 方塊註冊表。  目前註冊的方塊： - r_concrete: RC 混凝土方塊 (預設材料 STONE) - r_rebar:    鋼筋方塊 - r_steel:    結構鋼方塊 - r_timber:   木結構方塊  每個方塊都搭載 RBlockEntity，可獨立設定材料參數。 BlockItem 同步註冊，讓玩家可從創造模式取得方塊。
