# 🧩 com.blockreality.api.client.render.pipeline.BRViewportManager

> [!info] 摘要
> 多视口相机系统管理器  灵感来自 Create Aeronautics 和 Axiom，实现 Rhino 风格的多视图编辑功能。 支持单视口、双视口（水平/垂直）、四视口（QUAD）布局， 每个视口均有独立的 FBO、投影矩阵和相机参数。  正交视图自动同步到透视视图的目标位置， 提供完整的输入路由、FBO 管理和视口布局切换功能。  @author Block Reality Team @since 1.0.0
