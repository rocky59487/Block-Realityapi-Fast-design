# 🧩 com.blockreality.api.client.render.rt.BRVulkanDevice

> [!info] 摘要
> Manages Vulkan device initialization for the hybrid GL+VK ray tracing pipeline. Uses LWJGL Vulkan bindings. All operations are designed for graceful degradation: if Vulkan or RT extensions are unavailable, the system silently disables RT support without crashing the game.
