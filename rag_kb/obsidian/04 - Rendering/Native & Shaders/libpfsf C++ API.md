# 📄 libpfsf C++ API

## 📖 內容
libpfsf 提供獨立的 C 風格 API（pfsf.h / pfsf_types.h），包含 pfsf_create_context、pfsf_submit_island、pfsf_solve、pfsf_read_results 等函數。可用於離線 batch 或外部工具整合。
