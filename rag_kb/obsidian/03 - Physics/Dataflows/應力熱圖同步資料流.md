# 🌊 應力熱圖同步資料流

## 📝 概述
PFSFEngine 計算完成 → PFSFStressSyncPacket / StressSyncPacket 將應力峰值/閾值資料序列化 → BRNetwork 發送至所有觀察該區塊的客戶端 → 客戶端 ClientStressCache 接收並快取 → StressHeatmapRenderer / AnchorPathRenderer 在 RenderLevelStageEvent 中繪製熱圖或錨定路徑。
