# 🛠️ start-trainer 一鍵啟動 ML 訓練環境

## 📝 說明
提供 bat / sh / ps1 三種版本，會自動建立 Python venv、安裝 brml 依賴、啟動 UI。Windows: .\start-trainer.bat；Linux/Mac: ./start-trainer.sh。

> [!tip] 使用資訊
> ⌨️ Commands:
>   - `.\\start-trainer.bat`
>   - `./start-trainer.sh`
>   - `./start-trainer.ps1`
