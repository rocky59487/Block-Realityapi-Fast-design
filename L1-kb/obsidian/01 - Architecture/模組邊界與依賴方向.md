# 📄 模組邊界與依賴方向

## 📖 內容
關鍵依賴方向（絕不可違反）：fastdesign → api（單向）。api 不得引用 fastdesign 的任何類別。若需要在 api 中暴露擴展點，使用 SPI 接口。

> [!warning] 注意
> 違反此方向會導致編譯失敗或伺服器端 crash
