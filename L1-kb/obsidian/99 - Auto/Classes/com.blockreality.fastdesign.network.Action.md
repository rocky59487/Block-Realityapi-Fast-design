# 🧩 com.blockreality.fastdesign.network.Action

> [!info] 摘要
> Client → Server：Fast Design GUI 操作封包 — 開發手冊 §11.3  允許 FastDesignScreen 觸發伺服器端操作，無需輸入指令。  支援的操作： - UNDO：觸發還原 - SAVE：儲存藍圖（附帶名稱字符串） - LOAD：載入藍圖（附帶名稱字符串） - EXPORT：觸發 NURBS 匯出 - COPY：複製選取區域到剪貼簿 - PASTE：在玩家位置粘貼剪貼簿 - CLEAR：清除選取區域方塊 - SET_POS1：在玩家位置設定 pos1 - SET_POS2：在玩家位置設定 pos2
