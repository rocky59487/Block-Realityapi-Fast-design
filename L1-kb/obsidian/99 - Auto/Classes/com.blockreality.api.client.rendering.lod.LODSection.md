# 🧩 com.blockreality.api.client.rendering.lod.LODSection

> [!info] 摘要
> LOD 體素區段資料結構。  <p>代表一個 16×16×16 的 Minecraft section 在特定 LOD 等級下的 GPU 可用幾何。 同一個 section 可同時持有多個 LOD 等級的快取（0-3）， 依相機距離由 {@link BRVoxelLODManager} 選擇最合適的等級渲染。
