# 🧩 com.blockreality.fastdesign.client.node.panel.StylePreset

> [!info] 摘要
> 代表一個完整的光影「風格預設」，包含名稱、描述、縮圖資源、效能評級， 以及一組節點圖參數覆蓋值（portId → value）。  <p>StylePreset 本身是不可變資料物件；實際套用邏輯由 {@link StylePresetRegistry} 和 {@link BidirectionalSync} 負責。
