# 🧩 com.blockreality.api.physics.fluid.FluidType

> [!info] 摘要
> 流體類型列舉 — PFSF-Fluid 支援的流體種類。  <p>每種流體具有不同的密度和黏度，影響擴散速率和靜水壓計算。 當前版本僅完整實作 WATER，OIL/LAVA 預留供後續多相流體擴展。  <p>GPU shader 中使用 {@link #getId()} 作為 uint8 辨識碼。
