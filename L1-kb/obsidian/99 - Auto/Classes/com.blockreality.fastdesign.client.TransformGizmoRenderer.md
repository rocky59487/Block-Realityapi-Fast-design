# 🧩 com.blockreality.fastdesign.client.TransformGizmoRenderer

> [!info] 摘要
> 3D 變換操縱桿渲染器 — Fast Design v2.0 Feature 2  在選取框中心渲染 X(紅)/Y(綠)/Z(藍) 三軸箭頭 Gizmo。 讓玩家拖曳箭頭來移動選取區域或藍圖，取代在 ControlPanelScreen 中手動輸入偏移數值。  設計參考：Blender / Unity 的 Transform Gizmo + Axiom Mod 的世界內操縱桿。  操作方式： - 準心懸停在軸箭頭上 → 高亮顯示 - 左鍵拖曳 → 沿該軸移動選取框 - 滑鼠滾輪 → 沿該軸旋轉 90° (結合 Shift)
