# 🧩 com.blockreality.fastdesign.client.node.panel.StylePresetRegistry

> [!info] 摘要
> 集中管理所有內建與使用者自定義的 {@link StylePreset}。  <p>內建預設共 8 種風格，涵蓋從最輕量到旗艦光追的全光譜， 每種風格都透過 {@link StylePreset.Builder} 宣告完整的節點埠覆蓋值， 確保套用後的結果完全由節點圖驅動，不繞過渲染管線。  <p>使用者可透過「從目前設定儲存」功能新增自定義預設， 自定義預設標記 {@code userDefined = true}，允許從 UI 刪除。
