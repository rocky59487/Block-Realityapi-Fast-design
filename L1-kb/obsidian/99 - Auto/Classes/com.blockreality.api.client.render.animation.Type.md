# 🧩 com.blockreality.api.client.render.animation.Type

> [!info] 摘要
> 緩動函數庫 — GeckoLib 風格動畫插值。  提供 30+ 種緩動函數，涵蓋： - 線性、二次、三次、四次、五次 - 正弦、指數、圓形、彈性、回彈、反彈 - 每種含 easeIn / easeOut / easeInOut 三個變體  所有函數接受 t ∈ [0,1]，回傳 [0,1]。 無分配、無分支預測失敗 — 純數學運算。
