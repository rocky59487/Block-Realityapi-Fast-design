# 📄 C++ 獨立求解器（libpfsf）

## 📖 內容
libpfsf 是不依賴 Java/JNI 的獨立 C++ Vulkan Compute 求解器，包含 Jacobi、V-Cycle、phase-field 核心。可用於離線 batch 計算或外部工具整合。
