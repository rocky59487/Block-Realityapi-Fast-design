# 🌊 流體-結構耦合資料流

## 📝 概述
BRConfig.isFluidEnabled() == true → FluidRegionRegistry 掃描水域 → FluidGPUEngine / FluidAsyncCompute 執行 Jacobi 壓力求解（fluid_jacobi.comp.glsl）→ FluidPressureCoupler 將流體壓力轉換為結構荷載 → FluidStructureCoupler 延遲 1 tick 將荷載注入到 PFSF source buffer → PFSFEngine 計算結構應力 → 若結構破壞導致流體屏障失效，觸發 FluidBarrierBreachEvent。

> [!info] 備註
> 流體系統預設關閉，且流體-結構耦合有 1 tick 延遲（設計如此）
