# 📄 節點核心類別

## 📖 內容
BRNode 是所有節點的基類，定義 Port 與 evaluate() 方法。NodeGraph 管理連線與拓撲排序。EvaluateScheduler 依拓撲排序呼叫 evaluate()。NodeGraphIO 負責序列化/反序列化。fastdesign 的 BRNode 位於 fastdesign/client/node/BRNode.java，與 api 的 BRNode 是不同層級的擴展。
