# 📄 節點開發流程

## 📖 內容
1. 繼承 BRNode，定義 Port。2. 實作 evaluate()（由 EvaluateScheduler 依拓撲排序呼叫）。3. 在 NodeRegistry 註冊。4. 如需 runtime 綁定，實作 IBinder<T>。新增大量節點後忘記註冊可執行 python fix_registry.py。
