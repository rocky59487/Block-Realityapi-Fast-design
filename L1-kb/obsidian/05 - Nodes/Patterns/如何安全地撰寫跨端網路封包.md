# 🧩 如何安全地撰寫跨端網路封包

## 📝 說明
正確模式：封包類別（Packet）本身不能 import client-only 類別。在 handle() 或 onClient() 中，透過完整限定名稱（FQN）字串延遲取得 client 類別，或使用 DistExecutor.safeRunForDist()。錯誤模式：在封包類別頂部寫 import com.blockreality.api.client...
